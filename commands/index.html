# Npm / Yarn

To install packages in package.json use

npm install
yarn install

# Install

npm i -g <pkg...>
yarn global add <pkg...>

install package-name globally

npm i <pkg...> --save
yarn add <pkg...>

install in dependencies

npm i <pkg...> --save-dev
yarn add <pkg...> -D

install in development dependencies

npm i <pkg>@version
yarn add <pkg>@version

install at version

npm uninstall <pkg>
yarn remove <pkg>

uninstall

# List

npm ls (-g) (--depth=0)
yarn list (--depth=0)

list packages (globally) (by depth)

# Update

npm outdated
yarn outdated

list all outdated packages

npm update <pkg>
yarn upgrade <pkg>

update package

npm install -g npm-check-updates
ncu -u
npm update

to update to a new major version for all the packages, instead install npm-check-updates then run it before updating all packages.
